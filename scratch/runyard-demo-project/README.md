# Runyard Demo Project

Welcome to the Runyard demo! This workspace demonstrates the core developer environment capabilities of the Runyard agentic IDE.

## Features to Test
1. **File Explorer**: Browse files and folders reactively.
2. **Tab System**: Open files, AI Chat, and the Agent Manager in side-by-side splits.
3. **Interactive Terminal**: Run shells directly under the editor.
4. **Git Operations**: Stage, commit, and inspect logs.
